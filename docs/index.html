survfit함수와 객체

  • Surv 객체와 공변량을 formula 형태의 인자로 받아 Kaplan-Meier 또는 Cox 모델 기반 생존 확률을 저장한 객체를 반환
plot(survfit(km~1))
km_fit <- survfit(km~rai_recur$sex)
plot(km_fit)
summary(km_fit)
